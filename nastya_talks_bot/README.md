# Документация к телеграмм-боту для изучения английского языка по карточкам (далее - телебот):

## Название телебота: NastyaTalksBot

*Функционал:*

  - Команда "/start" при первом входе.
  
  Телебот предлагает случайное русское слово из базы данных и 4 варианта его перевода, 
представленные в виде кнопок.

  При нажатии на верный перевод телебот выдаёт сообщение "Отлично! (Слово и Перевод)", 
при неправильном переводе - "Допущена ошибка! Попробуй ещё раз вспомнить слово ..."

Кнопки Управления:

- "Практиковаться" - предлагает слова для изучения.

- "Дальше" - предлагает следующее случайное слово для изучения.

- "Добавить новое слово" - добавляет новое слово в базу данных, которое закрепляется за данным пользователем.
Необходимо ввести через тире русское слово и его перевод.

- "Удалить слово" - удаляет слово, закреплённое за данным пользователем.
Необходимо ввести русское (или английское) слово, которое хотите удалить.

- "Показать все слова" - показывает все слова, общие и индивидуальные для данного пользователя, доступные для тренировки.

- "Назад" - выходит из состояния ввода нового слова, если вы передумали.

## Установка и настройка
Чтобы запустить бота, выполните следующие шаги:

1. ### *Установка зависимостей: установите необходимые библиотеки из файла requirements.txt:*

*pip install -r requirements.txt*

2. ### *Настройка базы данных:*

*Создайте базу данных PostgreSQL* (имя базы данных указывайте в переменной *DB_NAME* в файле .env)

токен бота (переменная *BOT_TOKEN*), адрес бота в Телеграмм (*BOT_ADDRESS*), хост (*DB_HOST*), порт (*DB_PORT*), пароль (*DB_PASSWORD*) и название базы данных (*DB_NAME*) должны храниться в файле .env в корне проекта

*[Cхема базы данных](/application/db/db_scheme.png).*

1. ### *Запуск приложения: запустите главный скрипт:*

*main.py*

Таблицы БД будут созданы и заполнены начальным набором слов.
Чат-бот готов к работе.
